HTML

HTML är en samling av koder och symboler som utgör det vi ser på våra webbläsare.

Element

Element är dem olika "koder" som används som tillexempel "< p >" som används för att
skriva text, men även texten som skrivs ingår i det elementet. Varje element har också en slut tag
som i detta fallet hade skrivits " < /p >". Exit tag används för att avsluta ett element.

ID

ID är ett slags verktyg som man kan använda för att förändra hur ett element
kommer att se ut på din hemsida. Du använder det genom att tilldela
ett element ett ID. Sedan använder du det ID i en css för att modifiera det
till att se ut som du vill. Dock kan du endast ändra ett element åt gångenmed ett ID.

Class

En class fungerar likadant som ID men det är mer effektivt att använda då du kan
använda en class till att ändra flera saker samtidigt. Så om du vill ändra alla dina
div element samtidigt til lsamma sak så fungerar det.

Responsive

När HTML och css kan ändra storlek på hemsidan beroende på om du använder telefon eller
dator samt vilken telefon du använder så är den repsonsive.

Meta

Tagen meta används för att definera metadata och används inte i bodyn utan i head vilket
betyder att det itne visas på din hemsida. Metadata är data angående data. Metadata kan specifiera
teckenkodningen för HTML-dokumentet med mer.

http

Http står för Hypertext Transfer Protocol och är ett protokoll som överför hypertext
från webservern till webläsaren.

https

Https är nästan samma sak som http. Det står för Hypertext Transfer Protocol Secure
och den stora skillnaden mellan http och https är att den sistnämnda har ett extra lager
med högre säkerhet som krypterar infromationen som skickas över till din dator så att den blir svårare
för oönskade personer att komma åt.

Compressed data

När en fil eller annan data på datorn komprimeras för att ta upp mindre plats så kallas den för
compressed data. Detta eftersom att man använder förre bits i compressed data.

CDN

Content delivery network eller CDN är en samling av utspridda servrar som sparar kopior på data
för att man i vissa lägen skall få snabb åtkomst till dem från närmsta möjliga server.

Local bootstrap

Bootstrap är en förgjord css med mallar för allt mellan navbars och containers.
Det är även bra för att det automatiskt mobilanpassar din hemsida utan att du behöver göra någonting.

JQuery

JQuery är ett bibliotek med javascript som webbutvecklare kan få tillgång till för att
underlätta och utveckla deras hemsidor.

Flexbox

Flexbox är en endimisionell layout metod för att kunna ändra position horisontellt och vertikalt.
Flexboxen kommer automatiskt att fylla i eller passa in i större eller mindre områden.

Grid

Grid är en layout använd i HTML för att diveidera olika delar av sidan i "grids" vilket förenklar
positionering på din hemsida.
